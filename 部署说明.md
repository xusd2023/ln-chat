# LN 聊天系统 - 部署说明

## 文件结构

```
I:\文件\LN\
├── LN.exe          ← 服务器主程序（双击运行）
├── index.html      ← 前端页面（必须与 LN.exe 同目录）
└── data\           ← 数据目录（自动生成）
    ├── users.json
    ├── messages.json
    ├── groups.json
    └── ...
```

## 启动方法

**双击 `LN.exe`** 即可启动，控制台会显示：

```
===========================================
  LN 聊天服务已启动！
  本机访问:   http://localhost:3000
  局域网访问: http://192.168.3.38:3000
===========================================
```

## 局域网访问

同一局域网内的设备，在浏览器打开：
```
http://192.168.3.38:3000
```

> ⚠️ 如果无法访问，请检查 Windows 防火墙是否放行 3000 端口：
> 控制面板 → Windows Defender 防火墙 → 允许应用或功能通过 → 添加端口 3000

## 防火墙放行（如需要）

以管理员身份运行 PowerShell：
```powershell
netsh advfirewall firewall add rule name="LN Chat 3000" dir=in action=allow protocol=TCP localport=3000
```

## 重新打包

如果修改了 server.js 或 index.html，在 `I:\文件\LN\data\` 目录执行：
```cmd
.\node_modules\.bin\pkg.cmd server.js --target node18-win-x64 --output ../LN.exe
```

## 注意事项

- `index.html` 必须与 `LN.exe` 放在同一目录
- `data\` 目录会在首次运行时自动创建，存储用户、消息等数据
- 端口默认 3000，可通过环境变量 `PORT` 修改
